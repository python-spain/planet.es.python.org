El Atareao: Reemplazar palabras en varios documentos en Ubuntu con replace-me

Reemplazar palabras en varios documentos es una operación que suelo realizar de forma habitual. Así por ejemplo, cuando reutilizo parte de la estructura de una aplicación para hacer otra, solo tengo que sustituir el nombre de la aplicación en todos los documentos.

Sin embargo, realizar esta operación, es sin lugar a dudas, algo realmente tedioso, puesto que tienes que entrar en cada uno de los archivos en busca de la palabra que quieres reemplazar.

Pero, ¿y si pudieras realizar el reemplazo de esa palabra en todos los archivos de una sola vez? Pues esta es la razón de replace-me.

Reemplazar palabras en varios documentos en Ubuntu con replace-me

replace-me es una aplicación que te permite reemplazar palabras en varios documentos de texto en Ubuntu de forma simultánea.

Uso y funcionamiento

Esta aplicación se puede utilizar de forma independiente, o bien, como un complemento para Nautilus. De forma que seleccionando aquellos archivos que consideremos y accediendo al menú contextual esto se añadirán a la aplicación, donde solo nos quedará añadir la palabra que queremos reemplazar y pulsar el botón para ejecutar la acción.

Para iniciar la aplicación lo podemos realizar desde el Dash o Tablero en Ubuntu,

o bien, desde el menú clásico de Gnome, o desde cualquier desde los lanzadores mas habituales como Plank, Synapse, etc…

Mientras que para utilizarlo desde Nautilus, solo tenemos que seleccionar los archivos en los que queremos reemplazar la palabra y acceder a la aplicación desde el menú contextual, para reemplazar palabras en varios documentos,

Ya solo nos queda seleccionar la palabra que queremos sustituir, y evidentemente su reemplazo, y a continuación pulsar el botón para comenzar…

Instalación

Para instalar esta aplicación o complemento de Nautilus, solo tienes que ejecutar las siguientes órdenes en un emulador de terminal,

sudo add-apt-repository ppa:atareao/nautilus-extensions
sudo apt-get update
sudo apt-get install replace-me
